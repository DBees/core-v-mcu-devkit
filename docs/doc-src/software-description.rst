Overview of Software Support
----------------------------

The CORE-V MCU DevKit is supported by the following software elements:

- The full-featured CORE-V SDK which includes an Eclipse-based IDE and debug support
- OpenHW GCC compiler tool chain
- FreeRTOS operating system
- Device drivers structured on Amazon Web Service CommonIO
- Example application code
- Built In Self Test software
- Command Line Interface for low-level hardware debug

These features are described in turn below.
